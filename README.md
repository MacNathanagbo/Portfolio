# MacNathan Portfolio

Portfolio personnel de **MacNathan Eutyche AGBO**, conçu en **HTML**, **CSS** et **JavaScript vanilla**.

Le site met en avant :
- le profil professionnel
- le parcours en monétique
- les projets réalisés
- les domaines d'expertise
- les moyens de contact

## Aperçu

Le portfolio propose une interface moderne avec :
- un écran d'introduction animé
- un changement de langue `FR / EN`
- plusieurs thèmes de couleur
- une section projets avec aperçu détaillé en modale
- une section expertise animée
- une adaptation responsive pour desktop et mobile

## Lancer le projet

Aucune installation n'est nécessaire.

1. Ouvrir le dossier du projet.
2. Lancer `index.html` dans un navigateur.

Pour un meilleur rendu en développement, utiliser un serveur local comme :

```bash
python -m http.server 3000
```

Puis ouvrir :

```text
http://localhost:3000
```
